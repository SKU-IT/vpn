آموزش ساخت اتصال VPN در Windows

استفاده از اتصال آماده

  1. فایل SKU_L2TP.pbk را برای پروتکل L2TP/IPSec یا فایل SKU_PPTP.pbk را برای پروتکل PPTP اجرا کنید.
    استفاده از استاندارد L2TP/IPSec به علت امنیت بالاتر پیشنهاد می‌شود.
  2. مطابق تصویر زیر بر روی دکمه‌ی Connect کلیک کنید.
  3. در پنجره‌ی جدید، نام کاربری (User name) و رمز عبور (Password) خود را وارد و سپس روی گزینه‌ی Connect کلیک نمایید تا اتصال برقرار شود.
  4. پس از اتمام کار حتماً اتصال خود را قطع نمایید. برای قطع اتصال فایل مذکور را دوباره اجرا و بر روی دکمه‌ی Hang Up کلیک کنید.
